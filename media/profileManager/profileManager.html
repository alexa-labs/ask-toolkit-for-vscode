Profile manager

The profile manager guides you through logging in with your Amazon Developer Account and granting permission for the toolkit to access your Alexa Skills resources. The toolkit supports storing credentials under multiple profiles, useful if you are working across multiple developer accounts. The currently in-use profile is displayed on the toolkit status bar and can be changed at any time.

Once you select a new or existing profile, your web browser will be opened to a Login with Amazon prompt and you will be guided through the process of signing in and granting permission to the toolkit.
