September 2020 update

Welcome to the updated Alexa Skills Toolkit for September 2020 (version 2.1). This update introduces a number of new features, enhancements, and bug fixes.

What's new?

Local skill simulator

Invoke your skill

The newly introduced skill simulator allows skill builders to directly invoke their skills from the toolkit. You can send text requests to your skill and see the corresponding text responses, as well as the complete JSON request and response payloads. The simulator maintains the skill session with your skill just as a device would, so you can test the dialog flow.¬†If your skill supports multiple languages, select the language you want to test from the drop-down list.

Preview APL responses

The simulator also supports rendering Alexa Presentation Language (APL) responses if your skill returns them. These non interactive APL response previews let you see how the APL response will look on a variety of device shapes which can be selected.

Replay requests

In addition, the toolkit simulator allows you to export requests you've made to your skill to a file. This allows you to replay those requests at a later time or share them with other developers for testing.
