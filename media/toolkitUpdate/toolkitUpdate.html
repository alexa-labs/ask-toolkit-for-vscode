March 2021 update

Welcome to the updated Alexa Skills Toolkit for March 2021 (version 2.5). This update introduces a number of new features and enhancements.

What's new?

Support for virtual device registration (Beta)

The ASK Toolkit now supports device registration. Device registration is to set up a virtual Alexa device and register the device with your Amazon account. This feature is a prerequisite to access the enhanced Simulator, which supports APL user events and audio output. For details, see the Register device instructions to register/delete a device in ASK toolkit.



Support for APL user events in Simulator (Beta)

The ASK Toolkit now supports APL user events in the Simulator views. Please check the SendEvent command to learn more about APL user events. This feature is supported on MacOS and Windows platforms only.



Support for audio output in Simulator (Beta)

The simulator now supports audio output for the skill response. This feature is supported on MacOS and Windows platforms only.

Expose invocation name in Simulator

The simulator now exposes invocation name in the chat input, which helps developers to invoke skill.
